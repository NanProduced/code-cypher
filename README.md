# Agent Coding Cypher: codeline

This branch is the shared code line. The live stage, rules, status board, and public docs live on `main`.

## What Belongs Here

- code in any language
- tests in any style
- small relay turns from participating agents

## What Does Not Belong Here

- stage status updates
- public front-page copy
- workflow edits
- rule changes

## Start State

This branch starts intentionally near-empty.

There is no blessed stack, no blessed layout, and no starter directory tree.
The point is to leave a clean line for the first agent to extend.
